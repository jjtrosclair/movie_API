Welcome to myFlix, an app where you can access information about your favorite movies and the ones you want to watch next!

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies to the user /movies GET None A JSON object with info about all the movies
Get data about a single movie /movies/[title] GET None JSON object holding data about a single movie:

{
"Name": "Drowning by Numbers",
“Description”: “A British coroner and his son plays games with a family of three very dangerous women.”,
"Director": {“Name": "Peter Greenaway",
“Birth” : “1942”,
“Death”: “N/A”} },
“Genre”:{
“Name”: “Tragicomedy”,
“Description”: “A film that is contains elements of both drama and comedy.”}
Get data about a genre /movies/Genre/[name] GET None A JSON object with a description of a genre
Get data about a director /movies/Director/[name] GET None A JSON object with a information about a director e.g. { “name” :“Val Hoskins” { “bio” : “From San Fermino, she loved films as soon as she was old enough to understand words. She graduated from NYU in 1987 and had firsts etc...”, “birth-year” : “1987”, “death-year” : “N/A”}}
Allow a new user to register an account /users POST A JSON object containing a users information:
{"Username": "MovieFan123",
“Email”: “movies@gmail.com”,
“Password” : “admin1234”,
“Birthday” : “1/1/1991”} }		 A text message indicating that the account was successfully created
Allow users to update their user info /users/[username] PUT A JSON object containing a users updated information:
{"Username": "MovieFan123",
“Email”: “movies@gmail.com”,
“Password” : “admin1234”,
“Birthday” : “1/1/1991”} }		 A test message indicating that the information was successfully updated
Allow users to add a movie to a list of their favorites /users/[username]/Favorites/[title] POST None A message indicating the movie was added to the user's list of favorites
Allow users to remove a movie from their list of favorite /users/[username]/Favorites/[title] DELETE None A message indicating the movie was removed from the user's list of favorites
Allow users to deregister /users/[username] DELETE None A message confirming the user's account was deleted
